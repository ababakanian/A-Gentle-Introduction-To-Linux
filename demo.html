Part 2

A Short History

It started in 1965 at AT&T Bell Labs

Ken Thompson and Dennis Ritchie created an operating system called Unix, written in C

It was one of the best operating systems at the time Dennis Ritchie. Died Oct, 2011 was an American computer scientist. He created the C programming . Ken Thompson has worked at Google, where he co-invented the Go programming language.

In 1983 Richard Stallman started the GNU Project

His goal was to create a free UNIX-like operating system, so he started creating tools necessary to create an operating system.

Stallman launched the GNU Project, founded the Free Software Foundation, developed the GNU Compiler Collection and GNU Emacs, GNU Debugger and wrote the GNU General Public License (GPL). He finally created the GNU Hurd but it didn't get pick up by the masses, maybe because the linux kernel came before critic of restricted computer access in the lab, funded primarily by the DARPA. MIT's Computer Lab installed a password control system in 1977, he decrypted the passwords to re-enable anonymous access to the systems. This was the spark of his movement In 1980, he and others were refused access to the source code for the software of a newly installed laser printer, the Xerox 9700 because he wanted to modified the software like the previous laser printer so it electronically messaged a user when print job was done or printer was jammed.

In 1991 the Linux Kernel was created

Linus Torvalds started working on an OS as a hobby project, and he released the source code to the public so others can use it and make it better

A Finish Student named Linus Torvalds started creating a Minix clone, and he used the GNU tools to create it. In 1987 existed a simple operating existed named MINIX, meant to be used in universities and research. Minix license didn't allow its code to be modified or be reused. He only asked that improvements should be shared. GPL.

The rise of GNU/Linux

"Who would have thought even five years ago (1991) that a world-class operating system could coalesce as if by magic out of part-time hacking by several thousand developers scattered all over the planet, connected only by the tenuous strands of the Internet" -Eric S. Raymond
GNU/Linux is installed on millions of devices, ranging from big server banks to small devices.
